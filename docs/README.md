# Documentación de AtomOS

## Introducción
AtomOS es un sistema de diseño basado en Atomic Design.

## Estructura del proyecto
- **atoms**: componentes básicos.
- **molecules**: combinaciones de átomos.
- **organisms**: secciones completas reutilizables.
- **templates**: páginas con placeholders.
- **pages**: páginas reales.

## Buenas prácticas
- Usa HTML semántico y accesible.
- Reutiliza las variables CSS definidas en `css/base/variables.css`.
- Mantén los componentes pequeños y modulares.

## Ejemplo de composición
Combina átomos para formar moléculas y estas para construir organismos y páginas completas.

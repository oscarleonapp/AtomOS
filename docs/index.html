Documentación

Este sistema sigue la metodología Atomic Design. Encuentra cada nivel en su carpeta correspondiente.

  • Átomos – componentes básicos.
  • Moléculas – combinaciones de átomos.
  • Organismos – secciones reutilizables.
  • Plantillas – estructuras de página.
  • Páginas – ejemplos de uso.
